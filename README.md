
# Webstagram (인스타그램 클론코딩)
권준상


## 구현 사항

- CRUD, Media (게시물 업로드, 게시물 보기, 게시물 업데이트, 게시물 삭제)
- User (회원가입 / 로그인 / 로그아웃)
- Static (css / img 연결)
- 템플릿 상속 구현 (base.html)
- 피드 별로 작성자 띄우기 & 유저 별로 profile.html 에 피드글 모아주기
